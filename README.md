# Catálogo de Cursos - Demo de Coordinación UAM

Este repositorio contiene la mini aplicación "Catálogo de Cursos" desarrollada en **C# (.NET)**.

El objetivo central de este proyecto no fue solo el código, sino la **práctica de la coordinación y el uso de herramientas digitales**.

### Flujo de Trabajo y Colaboración

Nuestro equipo se centró en aplicar las siguientes metodologías y herramientas:

* **Trello (Tablero Kanban):** Utilizamos listas (Backlog, In Progress, Ready, Review/QA, Done) para gestionar y dar seguimiento al trabajo.
* **Gestión de Tareas:** Cada tarea en Trello incluyó un responsable, estimación, fecha límite y su Definition of Done (DoD).
* **Git y Ramas:** Aplicamos un flujo de trabajo basado en ramas, asegurando que cada funcionalidad fuera desarrollada en una rama separada y fusionada a 'main' mediante Pull Requests (PR).
* **Commits y Evidencia:** Se realizaron **commits claros** y el enlace al PR/commit se adjuntó a la tarjeta de Trello como evidencia al cerrar cada tarea.

## Logros del Proyecto (Alcance Técnico)

Esta mini aplicación de consola cumple con los siguientes requisitos:

* **Ejecución:** El código compila y ejecuta correctamente en un entorno .NET.
* **Listado y Filtro:** Muestra una lista estática de 3 cursos y permite realizar un filtro de búsqueda simple.
* **Documentación:** Incluye este `README` y comentarios internos para la documentación.

---

## Cómo Ejecutar la Aplicación

Sigue estos pasos para probar la aplicación en tu entorno:

1.  **Prerrequisito:** Asegúrate de tener instalado el **SDK de .NET**.
2.  **Navegar a la Carpeta:** Abre la terminal y navega a la carpeta principal del proyecto (`/CatalogoCursos`).
3.  **Ejecución:** Utiliza el comando `dotnet run`.
4.  **Prueba:** La aplicación solicitará un texto de búsqueda.
